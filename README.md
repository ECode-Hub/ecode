## 易码

> 更好的记忆你的代码

ECode 是一个让记忆代码变得简单的工具，它采用类似艾宾浩斯记忆法的方式来记忆代码，总会在最合适的时间点告诉你复习曾经学过的代码。相对于传统的学习方法，它更有效率，你可以大大减少你的学习时间，做到事半功倍。
